# Laplace Core 15 stable (?

Laplace is a Minecraft server management panel that integrates a backend API, WebSocket real-time communication, and a powerful TUI (Terminal User Interface).

## Highlight  


An outstanding foolproof system

## Quick Start

### Requirements
- Node.js (v16+)
- Java (Required to run Minecraft server core)

### Installation

```bash
npm install
```

and then

```bash
npx ts-node backend/server.ts
```
